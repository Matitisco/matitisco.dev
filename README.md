# Matitisco.dev

Portfolio one-page de Matias Jose Tiscornia, enfocado en AI automation, conversational systems y productos reales.

## Stack

- Vite
- React
- TypeScript
- CSS custom

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Deploy en Vercel

Framework preset: Vite

Build command:

```bash
npm run build
```

Output directory:

```bash
dist
```
